# GYM NOTES

**GYM NOTES** is a simple application for logging gym workouts. With it, users can easily log in, record their workouts, and track their progress.

## Features

- **User registration and login**: Implement user authentication for registration and login.
- **Workout log**: Users can add, edit, and view their workouts.
- **Clean and simple interface**: Design focused on usability.

## Technologies

- **Python**
- **Flask**
- **SQLite**
- **HTML and CSS**

##How To Use:

1. Clone the GitHub Repository using:

```
git clone https://github.com/Guilherme-Software/GYM-Notes.git
```

2. Install the requirements.txt using command:

```
pip install -r requirements.txt
```

3. Use the command:

```
Flask --app GYM run
```

**The app wil be available at http://127.0.0.1:5000**
